Simulations

There are currently called "simulations." Some of them are meant to be games (scuba), while others are meant to be one-off experiments with visualization (l-system). Some of them are kind of a game, but also an exercise in feature-building (estudioso).

Estudioso

"Estudioso" is an English-to-Spanish translation app that I use to practice learning Spanish phrases. Currently it offers fill-in-the-blank style prompts for testing on the gender of nouns and translation of basic sentences.

SCUBA

This is the game that best fits the idea behind OK Arcade. It takes initial inspiration from Asteroids, at least with the player motion. If you touch the eel, you lose. If you reach the cage, you win! In the future, touching the cage will free the crabs, making you a hero to some.

L-Systems

A graphical utility used to explore Lindenmeyer Systems and their many visual forms. This app is extremely early in development, so there isn't much to do at the moment. The goal is to allow the user to explore l-systems in two major ways: by interacting with UI that changes a single instance of an l-system, and choosing from a list of slight permutations from the current l-system.
